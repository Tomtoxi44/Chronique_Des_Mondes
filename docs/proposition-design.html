L'or devient l'accent des actions et états actifs ; le violet passe en secondaire. Fond très légèrement réchauffé (encre aubergine) : les cartes semblent moins froides.

Ce qui change concrètement

Accent principal → or/ambre (#d4a03c, hover #e8b84b) : boutons primaires (CTA), état actif de navigation, soulignés de titres, anneaux de focus. Le violet (#8b5cf6) reste l'accent secondaire (liens, surbrillances).

Fond réchauffé : #0f0f1a → #14111f (encre légèrement aubergine) ; cartes #1f1f35 → #211c33. Moins « bleu clinique », plus « chronique ».

Contraste texte : le gris atténué #94a3b8 → #b0a9bd pour repasser au-dessus du seuil WCAG AA sur fond sombre.

Coût d'implémentation faible : tout passe par variables.css (tokens sémantiques déjà en place). En pratique, on ne touche qu'une poignée de variables — aucun composant à réécrire.
NB : maquette statique d'illustration ; les emojis remplacent les icônes Bootstrap de l'app. Libre à toi de garder l'actuel — la base est déjà solide.
